## 1\. Qwen AI là gì? Tổng quan & lịch sử phát triển

### Tên gọi, nhà phát triển & ý nghĩa “Qwen”

**Qwen AI** chính là Tongyi Qianwen. Alibaba Cloud – bộ phận đám mây của tập đoàn Alibaba – phát triển từ năm 2023. “Tongyi” nghĩa là thông hiểu mọi ý nghĩa. “Qwen” nghĩa là nghìn câu hỏi. Tên gọi thể hiện rõ tham vọng: một AI trả lời sâu và nhanh.

### Kiến trúc Transformer + hybrid thinking modes (mới 2026)

Qwen dùng kiến trúc Transformer. Đến Qwen3.6-Plus, nó nâng cấp thành hybrid architecture: linear attention kết hợp sparse MoE. Điểm mới nhất là hybrid thinking modes. Chế độ “Thinking” suy nghĩ logic từng bước. Chế độ “Non-Thinking” trả lời nhanh trực tiếp. Mô hình linh hoạt hơn hẳn các LLM cũ.

## 4\. Qwen API & phát triển dành cho lập trình viên

### Qwen API là gì? OpenAI-compatible

Qwen API tương thích 100% với OpenAI SDK. Bạn chuyển code chỉ vài phút, không thay đổi logic.

### Cách lấy Qwen API Key miễn phí qua Model Studio

Vào Alibaba Cloud Model Studio. Kích hoạt dịch vụ. Tạo API Key ngay trong dashboard. Tài khoản mới được 1 triệu tokens miễn phí trong 90 ngày đầu.

**Bảng giá API Qwen3.6 (input/output token 2026)**

| Mô hình | Input (1M tokens) | Output (1M tokens) | Ghi chú |
| --- | --- | --- | --- |
| Qwen3.6-Plus | ~0.40 USD | ~1.20 USD | Rẻ hơn 60% so với bản cũ |
| Qwen3.6-Turbo | ~0.10 USD | ~0.30 USD | Phù hợp dự án lớn |

### Ví dụ code Python (Chat Completions)

```
import openai

client = openai.OpenAI(
    api_key="YOUR_QWEN_API_KEY",
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1"
)

response = client.chat.completions.create(
    model="qwen3.6-plus",
    messages=[{"role": "user", "content": "Viết function Python tính fibonacci"}],
    temperature=0.7
)

print(response.choices[0].message.content)
```

### Qwen Code CLI – công cụ terminal agent

Qwen Code CLI là agent command-line mã nguồn mở. Cài đặt bằng lệnh:  
`pip install qwen-code-cli`

Sau đó chạy `qwen-cli`. Benchmark Terminal-Bench 2.0 của Qwen3.6-35B-A3B đạt 51.5 – vượt nhiều mô hình mã nguồn mở khác.
